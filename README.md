# NEAT CPP

Neat implementation using cpp 

### Requirements
- C++ Compiler (G++)
- CMAKE

### How to run

**Compilation**
```bash
path/to/neat > mkdir build && cd build
path/to/neat/build > cmake ..
path/to/neat/build > make
```

**Run**
```bash
path/to/neat > ./build/main
# OR
path/to/neat/build > ./main
```
